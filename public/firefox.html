Firefox support

Firefox has experimental support for MIDI included since version 97 (I think). The process of enabling it is awkward, so the usage of Chrome or similar is still recommended.

However, if you are a die hard Firefox fan as I am, you can do this:

  1. Enter about:config in the address bar, accept all prompts
  2. search for «midi» and enable dom.webmidi.enabled
  3. Download this addon, accept all and install *

    4. * The addon does nothing but allows the configurator to actually even request MIDI handling. This is stupid, but this is per Firefox requirements. Sigh. I really hope they push this all to the stable release soon...

    Done? Try loading the configurator again. A relaunch of Firefox with Dobrynya connected may be required, as some versions have been reported to not support hot plugging.
