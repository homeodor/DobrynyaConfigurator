Browser not supported

MIDI Dobrynya uses Web MIDI standard to communicate with the browser, and the Configurator uses certain rather cutting-edge technologies to work. Your browser doesn’t support some of these.

Browsers built on the Blink engine (version 98+) as well as the Firefox browser (99+) are supported.

You may want to install a recent version of a compatible browser, for example,

Google Chrome · Microsoft Edge · Opera · Firefox
